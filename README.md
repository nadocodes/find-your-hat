# FindYourHat

### To Play Game:
Get Character '*' to reach the hat '^' without going out of bounds or falling the holes 'O'.
* node main.js
* Input controls:
    * U/u = Up ↑
    * R/r = Right →
    * D/d = Down ↓
    * L/l = Left ←
